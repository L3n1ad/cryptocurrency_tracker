# cryptocurrency_weekend_hw

The app is a crypto currency tracker. The site gets the information from https://www.coingecko.com/ it displays 100 crypto curenncies. The app feautres two searching options. The first one is a dropdown menu and the second one is a search bar where the user can start to type the name of the currency and the displayed list will be filtered through after every input to show all the crypto currencies matching with the user input. Users can also click on more details button on the right the see more detailed information about each crypto currency and on the bottom of that pop up window there is an option the show a diagram about how the currency price changed within the choosen time period.
